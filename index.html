Koseven is a PHP framework based on defunct Kohana 3.3.X .
Fully compatible with Kohana and updated to work with PHP7

Koseven is an elegant, open source, and object oriented HMVC framework
built using PHP7, by a team of volunteers. It aims to be swift, secure, and small.

Released under a BSD license, Koseven can be used legally for any open source,
commercial, or personal project.
